# VELORA

A refined wellness and leisure concept built with [Next.js](https://nextjs.org), React, TypeScript, and Tailwind CSS.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

To create a static export:

```bash
npm run build
```

The static files are output to the `dist/` directory.

## Project Structure

- `src/app/` — Next.js App Router pages
- `src/components/` — React components (including shadcn/ui)
- `src/hooks/` — Custom React hooks
- `src/lib/` — Utility functions
- `public/` — Static assets
